nggs

nggs is a collection of components for angular 1.x which I developed while working on my day-to-day projects, everything was made focusing in make my team's development faster and easier.
As I really enjoyed working on these components and loved the result. I hope you guys appreciate it.

Demos

nggs.https

A component for easy generation of methods for accessing REST APIs using angular's $http.

nggs.loader

A component that launches a loader automatically by intercepting all http requests made by angular's $http service or manually thru a factory.

nggs.modals

A component for generating modal using UI Bootstrap, Bootstrap and Font Awesome libraries.

nggs.props

A component for loading properties to the application, it works as well as a session object.

All my code and demos are written based on John Papa's Angular Style Guide, if you have some doubts about the demos's source code, take a look on it, it's a really amazing style guide.

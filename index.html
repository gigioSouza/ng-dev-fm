ng-dev-fm

ng-dev-fm is a framework for angular 1.x which I developed while working on my day-to-day projects, everything was made focusing in make my team's development faster and easier.
As I really enjoyed working on this framework and loved the result I decided to bring it to the community, I hope you guys appreciate it.

Demos

fm.https

A component for easy generation of methods for accessing REST APIs using angular's $http.

fm.loader

A component that launches a loader automatically by intercepting all http requests made by angular components($http, $resource), it also have some configurations.

fm.modals

A component for generating modal using UI Bootstrap, Bootstrap and Font Awesome libraries.

fm.props

A component for loading properties to the application, it works as well as a session object.

ng-dev-fm

ng-dev-fm is a framework I developed while working on my day-to-day job, everything was made focusing in make my team develop faster and easier.
As I really enjoyed working on this framework and loved the result I decided to make them more configurable before delivering to the community, as it was too much specific for my team's projects.

Demos

fm.https

A engine for easy generation of methods for accessing REST APIs using angular's $http.

fm.loader

A component that launches a loader automatically by intercepting all http requests made by angular components($http, $resource), it also have some configurations.

fm.modals

A engine for generating modal using UI Bootstrap, Bootstrap and Font Awesome libraries.

fm.props

A component for loading properties to the application, it works as well as a Session.
